# 开心小店段子网
    1.一个可以分享搞笑段子、搞笑图片、搞笑 动图的网站，
    2.在这里， 你可以发表你的搞笑 帖子，分享快乐给他人，
    3.你也可以尽情享受他人段子带来的快乐，喜欢就点赞，不喜欢就踩，或者就某 一条帖子展开评论， 
    4.你还可以关注其他用户，实时浏览他人动态。希望你能在这里获得快乐
技术点：python3+Flask+html+css+js+jquery+ajax+mysql
